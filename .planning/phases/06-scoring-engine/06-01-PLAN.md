Scoring weights from spec (MTCH-04 through MTCH-10):
  title_similarity : 0.30   (MTCH-04)
  skills_overlap   : 0.25   (MTCH-05)
  industry_match   : 0.15   (MTCH-06)
  company_size_match: 0.10  (MTCH-07)
  location_fit     : 0.10   (MTCH-08)
  recency          : 0.05   (MTCH-09)
  feedback_boost   : 0.05   (MTCH-10)

Signal computation rules (from CONTEXT.md decisions):
  title_similarity : cosine_sim(query_embedding, job["embedding"]); 0.0 if job has no embedding
  skills_overlap   : len(user_skills_set & job_skills_set) / len(job_skills_set); 0.0 if job_skills empty
  industry_match   : 1.0 if job.industry in profile.preferred_industries else 0.3;
                     0.3 if profile.preferred_industries is empty (no preference)
  company_size_match: 1.0 if profile.preferred_company_size == ANY or matches job.company_size;
                     0.4 otherwise
  location_fit     : normalize_location(job.location_raw) in normalized_pref_set -> 1.0;
                     job.location_raw normalizes to "remote" -> 1.0;
                     preferred_locations empty -> 1.0 (no filter);
                     otherwise 0.2
  recency          : max(0.0, 1.0 - days_since_first_seen / 30.0);
                     days_since_first_seen = (utcnow - first_seen_at).days
  feedback_boost   : company in liked_companies -> 1.0;
                     company in disliked_companies -> 0.0;
                     otherwise -> 0.5 (cold-start / neutral)  [MTCH-13]

    After all tests are RED, implement scorer.py:

    1. WEIGHTS dict (exact values, must sum to 1.0):
       WEIGHTS = {
           "title_similarity": 0.30,
           "skills_overlap": 0.25,
           "industry_match": 0.15,
           "company_size_match": 0.10,
           "location_fit": 0.10,
           "recency": 0.05,
           "feedback_boost": 0.05,
       }

    2. score_title_similarity(query_embedding: list[float], job_embedding: list[float] | None) -> float:
       - Return 0.0 if job_embedding is None or empty
       - Use numpy: a = np.array(query_embedding); b = np.array(job_embedding)
       - cosine_sim = np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b) + 1e-9)
       - Return float(np.clip(cosine_sim, 0.0, 1.0))

    3. score_skills_overlap(user_skills: list[str], job_skills: list[str]) -> float:
       - Return 0.0 if job_skills is empty
       - Lowercase both sides before set comparison
       - Return len(user_set & job_set) / len(job_set)

    4. score_industry_match(job_industry: str | None, preferred_industries: list[str]) -> float:
       - If preferred_industries is empty: return 0.3
       - If job_industry is None: return 0.3
       - Return 1.0 if job_industry.lower() in {i.lower() for i in preferred_industries} else 0.3

    5. score_company_size_match(job_size: str | None, preferred_size: str) -> float:
       - If preferred_size == "any": return 1.0
       - If job_size is None: return 0.4
       - Return 1.0 if job_size.lower() == preferred_size.lower() else 0.4

    6. score_location_fit(location_raw: str, preferred_locations: list[str]) -> float:
       - Import normalize_location from wekruit_matching.matching.filters
       - If preferred_locations is empty: return 1.0
       - If normalize_location(location_raw) == "remote": return 1.0
       - pref_buckets = {normalize_location(p) for p in preferred_locations}
       - If "remote" in pref_buckets: return 1.0
       - job_buckets = set of normalize_location tokens from location_raw (split on ";" then ",")
       - Return 1.0 if job_buckets & pref_buckets else 0.2
       - Note: reuse the _job_location_buckets logic from filters.py — either import the private
         function or duplicate the 10-line split logic inline

    7. score_recency(first_seen_at: datetime) -> float:
       - from datetime import datetime, timezone
       - days_old = (datetime.now(timezone.utc) - first_seen_at).days
       - Return max(0.0, 1.0 - days_old / 30.0)

    8. score_feedback_boost(company_name: str, liked: list[str], disliked: list[str]) -> float:
       - Case-insensitive comparison
       - If company_name.lower() in {c.lower() for c in liked}: return 1.0
       - If company_name.lower() in {c.lower() for c in disliked}: return 0.0
       - Return 0.5  # neutral / cold-start

    9. score_job(job: dict, profile: UserProfile, query_embedding: list[float]) -> dict:
       - Compute all 7 signals
       - signals dict: maps signal name -> float value
       - score = sum(WEIGHTS[k] * signals[k] for k in WEIGHTS)
       - Return {"score": round(score, 6), "signals": signals}

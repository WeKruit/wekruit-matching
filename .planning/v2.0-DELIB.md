## Non-Negotiable Architecture Decisions

These decisions are now reflected in code and planning artifacts:

- Supabase DB Webhooks are the user-sync mechanism.
- VALET remains the source of truth for users and receives zero code changes.
- The Mac Mini pipeline remains the source of truth for jobs and stays on the Mac Mini.
- The pipeline adds a sync-to-Firebase step after the daily run.
- Matching uses filter-first Firestore queries before cosine similarity.
- Embeddings are stored in Firestore as `number[]`.
- Firestore collection ownership remains prefix-scoped:
  - `platform-users`
  - `matching-jobs`
  - `matching-feedback`
  - `matching-saved-jobs`
- The Python 7-signal scorer is ported to TypeScript with identical weights and formulas.
